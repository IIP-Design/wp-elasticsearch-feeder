== Description ==

Creates REST api endpoints for each post type and indexes them into Elasticsearch.

== Installation ==

1. Upload `wp-es-feeder.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to Settings > WP ES Feeder
